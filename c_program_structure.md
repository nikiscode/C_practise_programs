//Structure of a C Program
A C program follows a fixed structure for proper execution.
1. Documentation Section
Used to describe the program.
Example: Program to print Hello World
2. Preprocessor Directives
Processed before compilation.
Example: #include <stdio.h>
3. Global Declarations
Variables declared outside the main function.
Example: int x;
4. Main Function
Execution of the program starts from main.
Example: int main() { return 0; }
5. Local Declarations
Variables declared inside a function.
Example: int a, b;
6. Executable Statements
Statements that perform actions.
Example: printf("Hello World");
7. User Defined Functions
Functions created by the programmer.
Example: void display() { print message }
Example Program
Include header file, define main function, print output, return 0.
